# ADR-0002：采用 local-first 增量同步

- 状态：已接受
- 日期：2026-07-15

## 背景

核心场景不要求秒级实时，但必须离线可用、零持续费用优先，并且不能因服务故障阻塞任务操作。

## 决策

- 本地 SQLite 是客户端唯一数据来源。
- 同步使用 Cloudflare Workers + D1 REST 增量接口，不使用 WebSocket。
- 客户端正文端到端加密，服务端只存密文。
- 用户通过设备配对加入同步空间，不建立传统账号。
- 夸克网盘只保存加密备份，不参与实时同步。

## 结果

服务端可以无状态扩缩，客户端离线体验稳定。代价是密钥恢复和冲突协议必须在首版完成；丢失全部设备及恢复材料时无法找回密文。
