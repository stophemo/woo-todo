# Woo Todo v0.1.0

首个可安装版本，面向 Apple Silicon Mac 与 Android 13 及以上设备。目标设备为 M4 MacBook Air（macOS Tahoe）和三星 Galaxy S23 Ultra（Android 16 / One UI 8）。

## 下载

- `Woo-Todo-v0.1.0-android.apk`：使用项目专用 Release 密钥签名的 Android 安装包。
- `Woo-Todo-v0.1.0-macos-arm64.zip`：Apple Silicon macOS 应用，使用 ad-hoc 签名，未经过 Apple 公证。
- `SHA256SUMS.txt`：两个安装包的 SHA-256 校验值。

Android 签名证书 SHA-256：`77d9b1ff936a9ea9da7ccae4360ede8f1b32b25761378826de7d812bccdba7f7`。

## 首版能力

- 每日、每周、每月、闲时任务与主线、支线、外传级别。
- 完成、Pass、历史记录、履约统计和重复任务。
- macOS 透明毛玻璃悬浮面板、置顶、鼠标穿透、全局快捷键和菜单栏入口。
- Android 今日桌面 Widget、任务编辑排序与每天 23:10 规划提醒。
- 无账号二维码配对、端到端加密增量同步、设备撤销。
- `.wootodo` 加密备份导入导出。

## 安装前须知

- 项目当前不提供公共同步服务器。需要跨端同步时，请先按仓库文档自行部署 Cloudflare Workers + D1；不部署时双端可独立离线使用。
- macOS 包没有 Apple Developer ID 签名和公证，首次打开可能需要在“系统设置 → 隐私与安全性”中手动允许。
- 首次安装 Android 正式包前如装过 Debug 包，请先导出备份再卸载 Debug 包；两者签名不同，不能直接覆盖安装。
- 目标真机的 Widget、通知时效、窗口层级和 24 小时耗电仍需实机验收。长期使用期间请定期导出加密备份。
