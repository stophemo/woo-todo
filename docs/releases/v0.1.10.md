# Woo Todo v0.1.10

这一版修正 macOS 显示设置与菜单布局，并为双端顶部模板增加按自然月和天显示的耗时、截止变量。

## 主要变化

### macOS 显示与菜单

- 修正标题、副标题模板输入框的宽度分配，变量按钮不再留下大块空白区域。
- 全局快捷键改用 macOS 原生菜单快捷键栏显示，与菜单项右侧对齐。
- “恢复可交互”只在鼠标穿透开启时显示，日常菜单更简洁。

### 自然月与天变量

- 新增 `{elapsedMonthsDays}`，按自然月和天显示从起始日算起的耗时，例如 `4个月零22天`。
- 新增 `{deadlineMonthsDays}`，按自然月和天显示距截止日的时间；过期后保留负号。
- 原有 `{elapsedDays}` 与 `{deadlineDays}` 继续显示纯天数，现有模板无需修改。
- macOS 与 Android 使用相同的月末收敛规则，覆盖 28、29、30、31 日、闰日、同日和过期边界。
