# Woo Todo v0.1.11

这一版补齐快速新增、局域网同步和显示配置同步，并把更新提醒改为非阻塞的入口提示。

## 主要变化

### 快速新增与显示设置

- macOS 快速新增现在可以直接设置任务级别、重复规则、提醒开关与提醒时间。
- 标题、副标题模板、起始日期和截止日期会在 macOS 与 Android 之间同步。
- 显示模板继续支持中文或英文星期、纯天数，以及“几个月零几天”的耗时和截止时间。

### 同一网络同步

- 新增 macOS 与 Android 的局域网同步首版，不需要自定义云端同步空间。
- Mac 可作为局域网同步主机，Android 在配对入口选择扫码或其他方式连接。
- Worker、坚果云 WebDAV 和局域网同步互斥，切换方式时不会让网络成为本地任务操作的前置条件。

### 更新与升级体验

- 新版本会定期自动检查，并只在菜单或设置入口提示“有新版本可用”，不弹出强制对话框。
- 用户仍可手动点击“检查更新”，也可以忽略当前提示继续使用。
- Android 使用同包名、同正式签名覆盖安装会保留任务、同步身份和配对状态，不需要重新配对；请勿先卸载、清除数据或使用不同签名覆盖。
