# Woo Todo v0.1.3

这一版补齐坚果云自动同步、双端任务通知、可自定义快捷键、纪念日计数与 Android 桌面组件，让日常使用不再依赖手工导入导出。任务仍优先写入本地 SQLite；断网时可以正常增删改，恢复网络后自动合并。

## 下载

- `Woo-Todo-v0.1.3-android.apk`：使用项目长期 Release 密钥签名的 Android 安装包。
- `Woo-Todo-v0.1.3-macos-arm64.zip`：Apple Silicon macOS 应用，使用 ad-hoc 签名，未经过 Apple 公证。
- `SHA256SUMS.txt`：两个安装包的 SHA-256 校验值。

## 主要变化

### 坚果云自动同步

- Android 与 macOS 可使用同一组坚果云 WebDAV 应用密码、同步空间 ID 和同步密钥自动同步。
- 本地修改会立即排队同步，应用启动、网络恢复和约 15 分钟兜底周期也会触发同步；网络不可用不阻塞本地操作。
- 每次修改保存为独立的 AES-256-GCM 密文对象，坚果云无法读取任务标题、日期、状态或提醒时间。
- 双端按操作 ID 幂等合并，删除墓碑阻止旧任务复活；畸形对象或路径会终止本轮同步并保留待重试操作，不会静默漏数据。
- 自建 Worker 与坚果云模式互斥；离线接力与恢复备份继续保留。

### 双端通知与纪念日

- 有日期的未完成待办可单独设置 `HH:mm` 提醒时间，提醒会随加密同步和恢复备份跨端保留。
- Android 在重启、系统时间变化、同步和恢复后重排提醒；macOS 使用系统通知中心调度。
- Android 13 及以上首次需要通知权限；拒绝后应用不会在每次启动时重复弹窗。
- 双端可自定义纪念日标题与起始日期，并显示“第 N 天”；起始日按第 1 天计算。

### macOS

- 四项全局快捷键均可自定义：默认 `Shift + Option + N` 新增、`Shift + Option + L` 显隐任务板、`Shift + Option + T` 置顶、`Control + Option + Space` 切换鼠标穿透。
- 移除任务板顶部冗余日期，整理任务分组、间距与状态呈现。
- 置顶、穿透和任务板显隐状态持久化，菜单栏始终保留恢复交互入口。

### Android

- 今日桌面组件改用 `RemoteViewsService` 读取本地仓储，可显示并操作当天待办。
- 坚果云设置页支持小屏滚动；同步、恢复和任务变更后会刷新组件与提醒。
- 数据库升级到版本 6，新增任务提醒字段、WebDAV outbox 和幂等应用记录。

## 升级前须知

- Android 正式包沿用既有签名，可从 v0.1.2 覆盖升级；不要用 Debug APK 覆盖正式安装。
- 升级前建议导出一份加密恢复备份并单独保管口令。
- 坚果云必须使用“安全选项”中生成的应用密码，不要填写账户登录密码；同步密钥必须在两端完全一致并自行备份。
- 首次启用通知后，请在两台设备分别确认系统通知权限。Widget、通知时效、窗口层级和后台耗电仍以目标真机行为为准。
- macOS 包仍未经过 Apple 公证，首次打开新版本可能需要在“系统设置 → 隐私与安全性”中手动允许。
