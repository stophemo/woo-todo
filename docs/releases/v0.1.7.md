# Woo Todo v0.1.7

这一版是 Android 启动闪退热修复。已配置同步的 Android 设备在创建带网络约束的后台同步任务时，会因缺少网络状态权限立即退出；本版本补齐对应权限并增加回归测试。

## 下载

- `Woo-Todo-v0.1.7-android.apk`：使用项目长期 Release 密钥签名的 Android 安装包。
- `Woo-Todo-v0.1.7-macos-arm64.zip`：Apple Silicon macOS 应用，使用 ad-hoc 签名，未经过 Apple 公证。
- `SHA256SUMS.txt`：两个安装包的 SHA-256 校验值。

Android 签名证书 SHA-256：`77d9b1ff936a9ea9da7ccae4360ede8f1b32b25761378826de7d812bccdba7f7`。

## 主要变化

### Android 启动热修复

- 声明 `android.permission.ACCESS_NETWORK_STATE`，满足 `JobScheduler` 网络约束的系统要求。
- 修复已配置 Worker 或 WebDAV 同步后，应用启动、编辑任务和小组件进程可能立即闪退的问题。
- 增加 Manifest 权限契约测试，防止后台同步任务与应用权限再次不一致。

### 延续 v0.1.6 功能

- 顶部显示模板支持英文星期、中文星期简称、完整日期拆分和起止日期变量。
- 新版本提示支持“更新”和“忽略此版本”，同时保留手动检查更新入口。

## 升级前须知

- Android 正式包沿用既有签名，可直接覆盖升级并保留本地任务和同步配置；不要先卸载旧版。
- macOS 包仍未经过 Apple 公证，首次打开新版本可能需要在“系统设置 → 隐私与安全性”中手动允许。
