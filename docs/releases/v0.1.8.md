# Woo Todo v0.1.8

这一版将双端更新提醒改为非打扰式菜单状态。后台发现新正式版后不再弹出对话框或抢占焦点，只在应用菜单中显示可用版本，由用户自行决定何时打开下载页。

## 下载

- `Woo-Todo-v0.1.8-android.apk`：使用项目长期 Release 密钥签名的 Android 安装包。
- `Woo-Todo-v0.1.8-macos-arm64.zip`：Apple Silicon macOS 应用，使用 ad-hoc 签名，未经过 Apple 公证。
- `SHA256SUMS.txt`：两个安装包的 SHA-256 校验值。

Android 签名证书 SHA-256：`77d9b1ff936a9ea9da7ccae4360ede8f1b32b25761378826de7d812bccdba7f7`。

## 主要变化

### 非打扰式更新提示

- Android 的“更多”菜单和 macOS 菜单栏菜单会显示“有新版本可用”及版本号。
- 自动检查只更新菜单状态，不再显示更新弹窗、抢占焦点或中断当前操作。
- 点击菜单中的版本提示才会打开 GitHub 下载页；不会静默下载、自动安装或绕过系统安全检查。
- 可用版本会在本地持久保留，应用重启或暂时离线后仍可查看；安装到对应版本后自动清除。
- 手动“检查更新”入口继续保留，已是最新或检查失败时提供结果反馈。

## 升级前须知

- Android 正式包沿用既有签名，可直接覆盖升级并保留本地任务和同步配置；不要先卸载旧版。
- macOS 包仍未经过 Apple 公证，首次打开新版本可能需要在“系统设置 → 隐私与安全性”中手动允许。
