# Woo Todo v0.1.9

这一版完善了升级后的同步体验和两端入口的可发现性。普通覆盖升级不会要求 Android 重新配对；未连接时，手机可以直接选择扫码、粘贴链接或手动填写坚果云配置；macOS 的任务窗口和设置入口也已明确分开。

## 主要变化

### Android 配对与升级

- 使用同包名、同 Release 签名覆盖升级时，保留本地 SQLite、Android Keystore、同步凭据和配对状态，无需重新配对。
- 启动读取同步凭据期间显示“正在读取同步配置…”，暂时禁用配对按钮，避免误报未配对。
- 未连接时点击“配对”可选择“扫描二维码配对”“粘贴配对链接”或“手动填写坚果云配置”。
- 粘贴链接仅在当前输入框中解析，不主动读取、持久化或记录剪贴板内容。

### macOS 菜单与设置

- 菜单栏提供独立的“任务详情与统计…”和“设置…”入口。
- 应用菜单提供标准“设置…”和 `Command + ,` 快捷键。
- 管理窗口侧栏分为“任务与统计”和“设置”，窗口标题明确包含任务、统计与设置。

### 文档与安全说明

- README 改为面向使用者的安装、配对、更新、数据和隐私入口。
- 安装、坚果云和 Worker 配对指南同步新的菜单路径及升级保留规则。
